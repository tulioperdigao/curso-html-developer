O que é HTML?

"A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento."*

NÃO é uma linguagem de programação

Não se pode dizer que se programa em HTML pois ela não desempenha esta função. Como o próprio conceito supracitado diz, é uma linguagem de marcação de Hipertexto, ou seja, ela aborda o conteúdo da página e o organiza para que possa ser utilizado em um site.

Portanto, o correto a se dizer é que se desenvolve em HTML.

Estrutura Básica

Voltar para Índice

Um programa em HTML consiste em basicamente três tags importantes em sua estrutura geral:

  1. < html > < /html >
  2. < head > < /head >
  3. < body > < /body >

Tag HTML

Tudo o que estiver dentro dessa tag será parte da página html.

Tag HEAD

Dentro dela estão os elementos que são pré-carregados para a página. Não são exibidos para o usuário.

Tag BODY

Esta é a parte que é exibida para o usuário, o conteúdo da página.
